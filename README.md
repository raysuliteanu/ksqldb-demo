# Getting Started
## Prerequisites
* Docker
* JDK 11+

To start ksqldb use `docker-compose up`
which will start Zookeeper, the Kafka broker and Ksqldb server.

To launch the CLI run `docker exec -it ksqldb-cli ksql http://ksqldb-server:8088`
